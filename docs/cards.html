Cards

A Card component resembles a playing card and contains short, related pieces of information. They are used either to group information, present a summary or a link to additional details. Below are a few types of available cards.

Cards with Badges

Here are some examples of cards with badges. Use the class card-badge in the badge element.
The badge can be positioned to the right of the card using class badge-right in the badge element. By default it is set to the left.

The following colors are available:

  1. Red: red-badge
  2. Default (gold): card-badge-def
  3. Grey: grey-badge
  4. Green: green-badge
