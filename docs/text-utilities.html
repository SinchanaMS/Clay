Text Utilities are made of typography that helps in making the text legible, readable, and appealing to the user.

Paragraphs

Paragraphs are written using p tags. These can be used for subtitles, descriptions, or plain texts.
We have 5 sizes available along with a grey variant. Use class p-xl for extra-large text, p-lg for large text, p-sm for small text, p-xs for extra small text, and p-grey for grey text. Adding no class results in the default text.
