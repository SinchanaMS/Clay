Sub-navigation

Sub-navigation refers to the navigation UI that helps users access lower-level categories in the site's information architecture.

Use the class sub-navbar in the <nav> tag, subnav in the sub-menu container tags. Use class subnavbtn in the button tag and finally subnav-content in the menu list.

Hamburger

The hamburger menu, or the hamburger icon, is the button in websites and apps that typically opens up into a side menu or navigation drawer. Change the size of the screen to check the links change into a hamburger.

Use the class hamburger-navbar in the <nav> tag and both the id and class as ham-list in the <ul> tag.
